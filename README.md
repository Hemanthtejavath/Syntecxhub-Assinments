# Syntecxhub-Assinments

This repository contains my Syntecxhub internship assignment projects built using simple HTML, CSS, and JavaScript.

## Projects

### 1. Image Gallery App

Folder: `ImageGallery`

Features:
- Responsive image gallery grid
- Images loaded from the free Picsum Photos API
- Hover zoom and fade effects
- Captions on hover
- Click image to open preview
- Blurred background lightbox view

### 2. Landing Page Website

Folder: `WebsiteLandingPage`

Features:
- Modern hero section
- Responsive navbar with smooth scroll
- Feature cards with icons
- Creative image and list section
- Call-to-action section
- Social media icons in footer
- Hover animations and mobile responsive layout
